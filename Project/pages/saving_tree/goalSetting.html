Why Use Goal-Setting

Goal setting is one of the most powerful tools we have at our disposal. Goal setting should be used because it provides you with purpose, giving you direction toward a better financial well being.

How to Create a Goal

First, think of something you want to acheive. Now apply SMART goal setting.

  • Specific
  • Measurable
  • Acheivable
  • Relevant
  • Time-bound

By using SMART goal setting, you can make sure your goals are reasonable and actually help you acheive something.
